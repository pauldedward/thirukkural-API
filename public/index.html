Get Started

Welcome to Ped's thirukkural API. An API developed for web developers to get thirukkural as response using various methods.

Special credits for DB : tk120404

Documentation

Note :

  • Replace {base url} with https://ped-kural.herokuapp.com
  • Replace any value inside {} with custom values.
  • API only has GET methods, meaning you cannot post or put vise versa.
  • Check spelling of Topics of thirukkural at tk120404
  • Topic names refer to transliterations of the thirukkural objects in the above mentioned Database. Check the above mentioned DB to know all the topic names.
  • To know more about thirukkural, visit Thirukkural wikipedia

Single kural

To get any single kural, just add kural number as a string in query parameter after the base url. Response will be a json object with various fields

syntax : {base url}/{kural number}

sample : https://ped-kural.herokuapp.com/1

Random kural

To get a random kural, just add the word random in query parameter after the base url. Response will be a json object with various fields

syntax : {base url}/random

sample : https://ped-kural.herokuapp.com/random

Random kural from a topic

To get a random kural from a particular topic(Athikaaram, Iyal, Paal), just add the name of the topic in query params after the base url. Response will be a json object with various fields

syntax : {base url}/{Topic name}

sample : https://ped-kural.herokuapp.com/Araththuppaal

All from a topic

To get all thirukkural from a particular topic, just add "/topic/" and topic name in query params after the base url. Response will be an array of json objects with various fields

syntax : {base url}/topic/{topic name}

sample : https://ped-kural.herokuapp.com/topic/Araththuppaal

From To

To get an array of continuous kural, just add "/from/" and the starting number of kural, then "/to/" and the ending number of kural in query params after the base url. Response will be an array of json objects with various fields

syntax : {base url}/from/{starting number}/to/{ending number}

sample : https://ped-kural.herokuapp.com/from/20/to/25

Selective kural

To get any number of desired kural, just add "/select/" and then add the kural numbers you need as an javascript array or as a comma separated string ("2,3,4") in query params after the base url. Response will be an array of json objects with various fields

syntax : {base url}/select/{kuralNumberArray} or {base url}/select/{kuralNumberString}

sample : https://ped-kural.herokuapp.com/select/1,40,1330 or "https://ped-kural.herokuapp.com/select/" + kuralNumberArray where kuralNumberArray is an Array.
